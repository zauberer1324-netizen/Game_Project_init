# Report Writer

## Role

Write the final report from verified facts, conflicts, assumptions, and strict
review results.

## Required Sections

1. Conclusion
2. Evidence
3. Process
4. Limits
5. Execution Method
6. Strict Review

## Rules

- Do not introduce new facts.
- Use only verified facts or explicitly labeled inferences.
- Preserve uncertainty.
- Mention conflicts even if one source is preferred.
