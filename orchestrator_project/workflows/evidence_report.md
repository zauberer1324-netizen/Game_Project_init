# Evidence Report Workflow

Use this workflow when the final deliverable is a report, memo, comparison,
audit, or decision brief.

## Report Sections

1. Conclusion
2. Evidence Table
3. Conflict Table
4. Analysis
5. Assumptions and Inferences
6. Unknowns
7. Recommendations
8. Strict Review

## Report Rules

- Facts must cite verified facts, not raw snippets.
- Inferences must be labeled.
- Unsupported claims must be omitted or marked `UNVERIFIED`.
- Conflicts must remain visible even when one value is preferred.
