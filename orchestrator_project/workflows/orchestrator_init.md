# Orchestrator Init Workflow

Use this workflow before specialist workstreams begin.

## Steps

1. Read project memory and Orchestrator memory.
2. Run a one-question-at-a-time alignment interview.
3. Separate accepted decisions, assumptions, open questions, and later decisions.
4. Propose updates for context, contracts, first PRD, workstream briefs, and Orchestrator memory.
5. Show planned file changes before editing.
6. Update files only after user confirmation.

## Output

- Direction summary
- Open questions
- Proposed memory updates
- Workstream readiness status
- Next recommended action
