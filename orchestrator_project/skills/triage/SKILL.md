# Triage

Use exactly one category role and one state role for each issue.

## Category Roles

- `bug`
- `enhancement`

## State Roles

- `needs-triage`
- `needs-info`
- `ready-for-agent`
- `ready-for-human`
- `wontfix`

When moving an issue to `ready-for-agent`, write an agent brief with current
behavior, desired behavior, key interfaces, acceptance criteria, and out of
scope.
