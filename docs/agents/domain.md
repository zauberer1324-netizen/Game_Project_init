# Domain Docs

This repo uses a split workspace:

- `orchestrator_project/` contains reusable AI operating rules.
- Root `CONTEXT.md` contains game-specific domain language.
- Root `docs/adr/` contains game-specific decisions.
- Root `docs/prd/` contains game-specific product requirements.
- Root `docs/issues/` contains game-specific implementation slices.

Before planning, coding, debugging, or reporting, read root `AGENTS.md`, then
use the relevant workflow and skill files under `orchestrator_project/`.
