# Asset Pipeline Contract

## Purpose

Keep generated and hand-authored assets consistent and traceable.

## Required Fields To Resolve

- Asset naming convention
- File formats
- Resolution rules
- Sprite sheet or atlas rules
- Animation frame rules
- Source/reference storage rules
- License and provenance tracking
- Import pipeline into `game_project/`

## Workstream Rule

Illustration and character workstreams should place drafts under their own
`artifacts/` folders. The Orchestrator promotes accepted assets into
`game_project/` after review.
