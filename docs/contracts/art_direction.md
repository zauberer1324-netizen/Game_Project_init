# Art Direction Contract

## Purpose

Keep character design, illustration, UI, and asset work visually consistent.

## Required Fields To Resolve

- Overall style
- Shape language
- Character proportions
- Color palette
- Lighting style
- Texture/detail level
- Animation feel
- UI visual tone
- Prohibited style directions

## Workstream Rule

Character and illustration workstreams may propose visual directions, but the
Orchestrator must approve changes before they become the project standard.
