# Art Output Rubric

Use this for visual direction, character design, illustration, UI art, sprites,
concept images, and asset proposals.

## Checks

- Does it follow `art_direction.md`?
- Does it state source/reference provenance where relevant?
- Does it identify whether the asset is concept, placeholder, or production candidate?
- Does it provide file naming and export expectations?
- Does it list animation, resolution, palette, and readability concerns?
- Does it avoid promoting drafts directly into `game_project/` without Orchestrator review?

## Output Should Include

- Visual intent
- Constraints followed
- Asset list
- Usage target
- Revision notes
- Approval needs
