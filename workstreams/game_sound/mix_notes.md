# Game Sound Mix Notes

## Purpose

Track mix, priority, ducking, variation, and accessibility concerns for proposed
gameplay, UI, and ambience sounds.

## Priority Notes

- Critical gameplay warnings should sit above BGM and ambience.
- UI confirmation sounds should be clear but not fatiguing.
- Ambience should support world mood without hiding player feedback.

## Open Mix Questions

- Which events are high-frequency enough to need variation?
- Which events need cooldown or anti-spam rules?
- Which audio cues require visual or haptic equivalents?
