# Data

This folder stores evidence artifacts when a workflow needs traceable source
material.

Use it for evidence-heavy game work, such as researching real-world fish,
crops, tools, factories, historical references, or public-domain sources.

```text
data/raw/        original source captures or raw files
data/extracted/  extracted candidate facts
data/verified/   verified facts or accepted references
data/rejected/   rejected or superseded evidence
```

A purely fictional game may not need this folder during early design.
